# AutoTotem

**AutoTotem** is a Minecraft mod for **1.21.0-1.21.9** that automatically equips a **Totem of Undying** in the player's off-hand when it’s not already there. This ensures the player is always ready to survive deadly situations, making PvE and PvP more streamlined.

---

## Features

- Automatically equips a Totem of Undying in the off-hand when it's empty
- Configurable delay for checks
- Toggle the mod on/off with a configurable key (default: `R`)
- Lightweight and simple to use

---

## Installation

1. Download the **AutoTotem mod JAR** from the [Releases](https://github.com/your_username/your_repo/releases) section.
2. Place the JAR file in your **mods folder** in `.minecraft/` for Fabric 1.21.0-1.21.9.
3. Launch Minecraft with the **Fabric loader**.

---

## Configuration

- The mod is **enabled/disabled** with a configurable keybind (default: `R`).
- **Tick delay** can be set in the config file to control how often the mod checks the off-hand.

---

## Usage

1. Once installed, press the **toggle key** (`R` by default) to enable or disable the mod.
2. When enabled, the mod will **automatically** equip a Totem of Undying to the off-hand if it's missing.
3. **Keybindings** can be customized from the Minecraft options.

---

## Troubleshooting

- **The mod doesn’t work:**  
  Make sure the mod is placed correctly in the `mods` folder and that you're running Minecraft with the Fabric loader. Check the version compatibility with your Minecraft version.

- **The totem isn’t equipping:**  
  Check the **config file** to ensure the mod is enabled and the correct delay is set. Verify your keybinding settings in Minecraft.
